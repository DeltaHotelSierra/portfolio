Plan for Digital Safety

Mission

Creating accessible cybersecurity education and practical safety measures for individuals and organizations to protect themselves in an increasingly digital world.

Core Safety Principles

1. Password Security

  • Use strong, unique passwords for every account
  • Implement password managers (Bitwarden, 1Password)
  • Enable two-factor authentication (2FA) everywhere possible
  • Never reuse passwords across different services

2. Network Protection

  • Use VPNs on public Wi-Fi networks
  • Secure home networks with strong encryption (WPA3)
  • Regularly update router firmware
  • Disable unnecessary network services

3. Device Security

  • Keep all software and operating systems updated
  • Use reputable antivirus and anti-malware solutions
  • Enable full-disk encryption
  • Regular backups (3-2-1 backup rule)

4. Online Behavior

  • Be cautious of phishing emails and suspicious links
  • Verify website authenticity before entering credentials
  • Limit personal information shared on social media
  • Use privacy-focused browsers and extensions

Implementation Roadmap

This project aims to develop comprehensive guides, tools, and resources to help people implement these safety measures step-by-step, making cybersecurity accessible to everyone regardless of technical expertise.
